# 🚀 CareerPilot AI

CareerPilot AI is a beautiful AI-inspired web application that generates personalized career roadmaps for students and professionals.

No backend.
No API keys.
Runs completely in your browser.

---

## ✨ Features

🎯 Career Selection

📅 Personalized Learning Roadmap

📚 Skill Recommendations

💼 Internship Roadmap

🏆 Readiness Score

📈 Progress Dashboard

📥 Download Career Plan

💾 Save History

🌙 Dark / Light Mode

📱 Fully Responsive
